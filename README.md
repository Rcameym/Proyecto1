//TimelineMax: funcionalmente similar a TimelineLite pero con agregados como yoyo, las «callback functions» personalizadas y más. Es genial para secuencias avanzadas de múltiples interpolaciones y líneas de tiempo.
plugins/scrollmagic/ScrollMagic.min.js
//Requiere la biblioteca jQuery. Este complemento está diseñado para usarse junto con jQuery. Permite a ScrollMagic hacer uso del motor de selección avanzado de jQuery (chisporroteo) para todos los elementos suministrados a los objetos ScrollMagic, como contenedores de desplazamiento o elementos de activación. ScrollMagic también acepta elementos jQuery para todos los métodos que esperan referencias a elementos DOM. Tenga en cuenta que en la mayoría de los casos se utilizará el primer elemento del conjunto emparejado. Además, proporciona el objeto ScrollMagic dentro del espacio de nombres jQuery, por lo que se puede acceder a él usando '$ .ScrollMagic'.
plugins/greensock/animation.gsap.min.js
//Permite almacenar la referencia a la animación a una secuencia, para poder trabajar con ella desde diferentes puntos del código. Capacidad de asignar una función callback al completar la animación (también el paso de parámetros a la misma).
plugins/greensock/ScrollToPlugin.min.js
//Anima la posición de desplazamiento de la ventana (como hacer window.scrollTo(x, y)) o un elemento DOM (como hacer myDiv.scrollTop = y; myDiv.scrollLeft = x;).

//Para animar anchors en un sitio web, para que hubiera un deslizamiento cuando baja o sube la página.

//Su nombre es Easing jQuery Plugin y está ampliamente extendido en el mundillo de los widgets jQuery. Es un componente tan útil como versátil que nos permite implementar un gran abanico de funciones de efectos de animación en jQuery

Configuración de una Red Lan

Este proyecto formo parte del curso Redes de la facultad de Ingeniería en Sistemas de la Información de la Universidad Mariano Gálvez, se configuro por medio de Windows Server los servicios: Active Directory, FTP, Firewall, Servidor DHCP, Servidor IIS, Servidor DNS. Se crearon dos diferentes vlan´s con la ayuda del software Putty, donde cada vlan tenía un segmento diferente en dos switch administrables. Cada host podía tener comunicación con diferente segmento, pero cada interfaz de los switch, solo podían reconocer una mac-addres, por lo que necesitaba apoyo para cambiar la configuración si trasladaban un host a una diferente interfaz. Todos los host tenían acceso a los servicios configurados en Windows Server.

Creación de Base de Datos Homogenea

Fue para el curso de Base de Datos II de la Facultad de Ingeniería en Sistemas de la Información de la Universidad Mariano Gálvez. En esa base de datos todos los sitios tienen idéntico software de sistemas Gestores de bases de datos, son conscientes de la existencia de los demás sitios y acuerdan cooperar en el procesamiento de las solicitudes de los usuarios. En estos sistemas los sitios renuncian a una parte de su autonomía en cuanto a su derecho de modificar los esquemas o el software gestor de base de datos. Ese software también debe cooperar con los demás sitios en el intercambio de la información sobre las transacciones para hacer posible el procesamiento de las transacciones entre varios sitios. Se utilizo el software sqldeveloper para dicho proyecto.
